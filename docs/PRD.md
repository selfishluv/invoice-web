# 아이타임 (AiTime) PRD

## 핵심 정보

**목적**: PostgreSQL DB를 데이터 저장소로 활용하여 아이의 성장 기록을 타임라인 형태로 웹에서 볼 수 있게 한다
**사용자**: 아이의 성장 기록을 디지털로 보관하고 싶은 부모

---

## 사용자 여정

```
1. 타임라인 페이지 (메인)
   - 사이트 접속 시 자동으로 타임라인 페이지 표시
   - 전체 성장 기록이 날짜 역순으로 카드 형태로 나열됨
   ↓ 태그 칩 클릭

2. 태그 필터 적용 상태
   - 선택한 태그에 해당하는 기록만 표시
   - 여러 태그 동시 선택 가능 (OR 방식)
   - 태그 재클릭 시 필터 해제
   ↓ 기록 카드 클릭

3. 상세 보기 모달
   - 사진 확대 + 전체 메모 표시
   - 모달 닫기 → 타임라인으로 복귀
   ↓ 모달 닫기

4. 타임라인 페이지 복귀
   - 기존 필터 상태 유지
```

## 데이터 모델

### GrowthRecord 인터페이스 (TypeScript)

```typescript
interface GrowthRecord {
  id: string // cuid
  title: string // 이벤트 제목
  date: string // YYYY-MM-DD (ISO 날짜 문자열)
  photoUrl: string | null // 외부 이미지 URL
  tags: string[] // 이정표 태그 목록
  memo: string // 상세 메모/기록
  ageMonths: number | null // 개월 수
}
```

### Prisma 스키마 (prisma/schema.prisma)

```prisma
model GrowthRecord {
  id        String   @id @default(cuid())
  title     String
  date      DateTime
  memo      String   @default("")
  photoUrl  String?
  tags      String[]
  ageMonths Int?
  createdAt DateTime @default(now())
  updatedAt DateTime @updatedAt
}
```

### 태그 시스템 (src/lib/types/tag.ts)

미리 정의된 태그 색상 매핑 (`TAG_COLORS`) 기반으로 렌더링:
`첫걸음`, `첫말`, `첫니`, `생일`, `병원`, `여행` (확장 가능)

## 기술 스택

### 프론트엔드 프레임워크

- **Next.js 15.5.3** (App Router + Turbopack)
- **TypeScript 5**
- **React 19.1.0**

### 스타일링 & UI

- **TailwindCSS v4** (설정 파일 없는 새로운 엔진)
- **shadcn/ui** (new-york style) — Dialog, Badge, Skeleton, Card 등
- **Lucide React** — 아이콘
- **tw-animate-css** — 애니메이션

### 데이터 레이어

- **PostgreSQL** — 관계형 DB (성장 기록 저장)
- **Prisma ORM** (`prisma` + `@prisma/adapter-pg`) — 타입 안전한 DB 접근
- **ISR (Incremental Static Regeneration)** — DB 데이터 캐싱 (revalidate: 3600)

### 배포 & 호스팅

- **Vercel** — Next.js 15 최적화 배포 플랫폼

### 패키지 관리

- **npm**
